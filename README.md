### Dependenacy and Package Management
ABCD

The following allows me to utilize a *Dependency Checker* to sort out unecessary libraries and packages my Portfolio may not need.
```bash
depcheck
```
When the above command is ran in the **Terminal**, I am presented with a list of React NPM Dependencies I can prune. Using this allows me to execute the following command and clean my Project.
```bash
npm uninstall <package-name>
```
